Antes de escrever qualquer linha, o documento precisa estar configurado corretamente
📖
O que é ABNT?
ABNT é a Associação Brasileira de Normas Técnicas. A norma ABNT NBR 6022 define como artigos acadêmicos devem ser formatados no Brasil. Faculdades, ETEC e universidades exigem esse padrão.

📏

Siga as instruções para o seu aplicativo:

1
Abrir o Word e criar novo documento
Abra o Microsoft Word. Clique em Arquivo → Novo → Documento em Branco. Ou pressione o atalho Ctrl + N.
2
Salvar o arquivo com nome correto
Pressione Ctrl + Shift + S para "Salvar como". Escolha uma pasta. No campo Nome do arquivo digite: Artigo_GenerosJogos_SeuNome. Clique em Salvar.
⚠️ Salve sempre em uma pasta que você saiba encontrar depois. Recomendamos criar uma pasta chamada "PTIC" em Documentos.
3
Configurar as margens ABNT
Clique na aba "Layout" (ou "Layout da Página"). Clique em "Margens" → "Margens Personalizadas...". Preencha: Superior: 3cm | Esquerda: 3cm | Inferior: 2cm | Direita: 2cm. Clique em OK.
Configuração de Margens:
Superior: 3 cm  |  Inferior: 2 cm
Esquerda: 3 cm  |  Direita: 2 cm
4
Configurar a fonte Times New Roman 12
Pressione Ctrl + A para selecionar tudo. Na aba "Início", no campo de fonte, selecione Times New Roman. No campo do tamanho, selecione 12.
5
Configurar espaçamento 1,5
Com tudo selecionado (Ctrl+A), clique com o botão direito → "Parágrafo". Em "Espaçamento entre linhas" selecione 1,5. Em "Recuo especial" selecione Primeira linha: 1,25 cm. Clique OK.
✅ Pronto! Seu documento está configurado no padrão ABNT. Agora é hora de escrever.
1
Criar um novo documento no Google Docs
Abra o navegador e acesse docs.google.com. Clique no grande "+" (Documento em branco).
2
Nomear o documento
Clique em "Documento sem título" no canto superior esquerdo. Digite: Artigo_GenerosJogos_SeuNome. O Google Docs salva automaticamente!
✅ O ponto verde animado na barra superior confirma que está salvando em tempo real.
3
Configurar as margens ABNT
Vá em Arquivo → Configurar Página. Preencha: Superior: 3cm | Esquerda: 3cm | Inferior: 2cm | Direita: 2cm. Clique em OK.

Esquerda: 3 cm  |  Direita: 2 cm

Configurar fonte e espaçamento
Pressione Ctrl + A para selecionar tudo. Na barra de ferramentas, clique no nome da fonte (geralmente "Arial") e troque para Times New Roman. Troque o tamanho para 12. Para o espaçamento: menu Formatar → Espaçamento entre linhas → 1,5.
✅ Documento configurado! O Google Docs já está pronto para receber seu artigo.

Aplicando Estilos — a base de tudo
Sem estilos, não existe sumário automático. Esse é o passo mais importante!
🎯
Por que usar estilos?
Um estilo é uma receita de formatação guardada com um nome. Quando você aplica "Título 1", o Word/Docs já sabe que aquilo é uma seção principal e vai incluir no sumário automaticamente. Se precisar mudar a aparência de todos os títulos, basta mudar o estilo uma vez!

Formatando as Referências
A seção REFERÊNCIAS lista todas as fontes usadas — é obrigatória e tem formato próprio
⚠️
Atenção: Referências ≠ Bibliografia
Referências listam APENAS as fontes que você citou no texto. Coloque em ordem alfabética pelo sobrenome do autor. Espaçamento simples (1,0) entre as linhas de cada referência e linha em branco entre elas.

Modelos de referência ABNT — copie e adapte:

TIPO: Site / Página Web
NOME DO ESTÚDIO. Título da página. Local, ano. Disponível em: <URL>. Acesso em: dia mês ano.

Exemplo:
CD PROJEKT RED. The Witcher 3: Wild Hunt. Polônia, 2015. Disponível em: <https://www.cdprojektred.com>. Acesso em: 20 jun. 2026.
TIPO: Artigo Acadêmico
SOBRENOME, Nome. Título do artigo. Nome da Revista, Cidade, v. X, n. X, p. XX-XX, mês ano.

Exemplo:
SILVA, João. Gêneros de jogos digitais e design de interação. Revista Brasileira de Games, São Paulo, v. 5, n. 2, p. 30-45, jan. 2023.
TIPO: Livro
SOBRENOME, Nome. Título do Livro. Edição. Cidade: Editora, ano.

Exemplo:
SCHELL, Jesse. A Arte de Game Design: O Livro dos Lenses. Rio de Janeiro: Elsevier, 2011.

☁️
Exportar do Word para Google Docs e vice-versa
Aprenda a trabalhar nos dois formatos — habilidade essencial no mercado de trabalho
1
Salvar o arquivo Word (.docx)
Pressione Ctrl + S no Word. Certifique-se que o arquivo está salvo como .docx (formato padrão do Word). Verifique o nome do arquivo na barra de título.
2
Fazer upload para o Google Drive
Acesse drive.google.com. Clique em "+ Novo" → "Upload de arquivo". Selecione o arquivo .docx do seu artigo. Aguarde o upload.
3
Abrir no Google Docs
No Drive, clique com o botão direito no arquivo → "Abrir com" → "Google Docs". O arquivo abre no Docs mantendo a formatação!
⚠️ Algumas formatações avançadas podem não ser mantidas perfeitamente. Revise o documento após a conversão.
4
Compartilhar com o professor
Com o arquivo aberto no Google Docs, clique em "Compartilhar" (botão azul, canto superior direito). Adicione o e-mail do professor. Selecione permissão "Comentarista". Clique em "Enviar".
✅ O professor poderá ver e comentar o documento sem precisar baixá-lo!
1
Baixar como Word (.docx) do Google Docs
Com o documento aberto no Google Docs, vá em Arquivo → Fazer download → Microsoft Word (.docx). O arquivo será baixado para a pasta Downloads.
2
Baixar como PDF
Vá em Arquivo → Fazer download → Documento PDF (.pdf). O PDF é ideal para enviar por e-mail, pois preserva toda a formatação em qualquer computador.
💡 Sempre entregue seu artigo também em PDF quando possível — assim a formatação nunca se perde!
3
Obter link para compartilhamento
Clique em "Compartilhar" → "Copiar link". Altere o acesso para "Qualquer pessoa com o link" como Leitor. Cole o link na entrega da atividade.
