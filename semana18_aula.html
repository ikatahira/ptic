Estrutura Sequencial — A Base de Tudo

➡️
A estrutura sequencial é a mais simples: as instruções são executadas uma por uma, na ordem em que aparecem, de cima para baixo.

Exemplo em pseudocódigo:
INICIO
LEIA comprimento, largura
area = comprimento * largura
perimetro = 2 * (comprimento + largura)
ESCREVA "Área:", area
ESCREVA "Perímetro:", perimetro
FIM

Regra fundamental: a ordem importa! Se você calcular perimetro antes de ler os valores, o programa vai usar lixo de memória. Em programação, a sequência é lei!

Variáveis e Operadores — Blocos Básicos dos Algoritmos

🔢
Variável é um espaço na memória do computador com um nome e um valor que pode mudar.

Tipos de variáveis:
• Inteiro: números sem decimal (ex: 5, -3, 100)
• Real/Float: números com decimal (ex: 3.14, 9.8, -0.5)
• Texto/String: palavras e frases (ex: "Olá", "João")
• Lógico/Booleano: só Verdadeiro ou Falso

Operadores matemáticos:
+ soma | - subtração | * multiplicação | / divisão | % resto | ** potência

Atribuição: x = 5 significa "a variável x RECEBE o valor 5".
⚡ +10 XP

Entrada e Saída de Dados

📐
Todo algoritmo tem pelo menos uma entrada (dados que recebe) e uma saída (resultado que produz).

Em pseudocódigo:
LEIA nome, idade ← entrada (usuário digita)
ESCREVA "Olá,", nome ← saída (mostra resultado)

Em linguagens reais:
• Python: nome = input("Seu nome: ") e print("Olá,", nome)
• JavaScript: let nome = prompt("Seu nome:") e alert("Olá " + nome)

Dica importante: variáveis que vêm do usuário (LEIA) são a entrada do seu programa. Variáveis calculadas que você mostra (ESCREVA) são a saída.
⚡ +10 XP
✏️ Comando que exibe um resultado para o usuário em pseudocódigo é:
ESCREVA

Algoritmos de Cálculo — Sequencial na Prática

🧮
Algoritmos sequenciais são perfeitos para cálculos matemáticos simples.

Exemplo — Calcular troco:
INICIO
LEIA preco_produto, valor_pago
troco = valor_pago - preco_produto
SE troco >= 0 ENTAO
ESCREVA "Troco: R$", troco
SENAO
ESCREVA "Valor insuficiente!"
FIM

Calcular média: media = (n1 + n2 + n3) / 3
Calcular desconto: preco_final = preco * (1 - desconto/100)
Converter temperatura: celsius = (fahrenheit - 32) * 5 / 9
⚡ +10 XP
