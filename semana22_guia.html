📝 Guia de Exercícios — Semana 22
✨ LLMs e Prompt Engineering 3º Bimestre 3 exercícios

Este guia contém as instruções passo a passo, a rubrica de avaliação detalhada e os critérios de qualidade para cada exercício da Semana 22. Siga os passos na ordem indicada e use a rubrica para autoavaliar seu trabalho antes de entregar.

1
Dominar Prompt Engineering
📝 Prompts ⏱ 35 min

Crie prompts profissionais usando os 5 elementos e as técnicas Zero-Shot, Few-Shot e Chain-of-Thought.

📋 Passo a Passo
  1. TAREFA 1 — Prompt com os 5 elementos para criar uma redação escolar
  2. TAREFA 2 — Prompt Few-Shot: forneça 2 exemplos de transformação formal→informal, depois peça a transformação de um texto novo
  3. TAREFA 3 — Prompt Chain-of-Thought para um problema de lógica: 'Pense passo a passo antes de responder'
  4. TAREFA 4 — Comparação: faça a mesma pergunta com prompt simples e com prompt dos 5 elementos
  5. Documente: qual técnica gerou melhor resultado para cada tipo de tarefa?
✅ Rubrica de Avaliação (Total: 100 pts)
  • 4 prompts criados e testados (30 pts)
  • 5 elementos aplicados corretamente (20 pts)
  • Few-Shot com exemplos relevantes (20 pts)
  • Chain-of-Thought aplicado em problema lógico (20 pts)
  • Comparação documentada com análise (10 pts)
2
Detectar Alucinações da IA
🛡️ Verificação ⏱ 25 min

Treine seu senso crítico verificando sistematicamente afirmações geradas pela IA em fontes confiáveis.

📋 Passo a Passo
  1. Peça ao ChatGPT: '5 fatos históricos interessantes sobre o Brasil'
  2. Para CADA fato: pesquise no Google/Wikipedia para verificar precisão
  3. Classifique: Verdadeiro / Falso / Parcialmente verdadeiro / Não verificável
  4. Repita com um segundo tema (ciência, tecnologia, esportes, geografia)
  5. Calcule: qual % das afirmações foi precisa?
  6. Documente o fato mais incorreto encontrado com a versão correta
✅ Rubrica de Avaliação (Total: 100 pts)
  • 10 fatos verificados (2 temas × 5 fatos) (30 pts)
  • Verificação em fontes confiáveis documentada (25 pts)
  • Classificação correta de cada afirmação (25 pts)
  • Percentual de precisão calculado (10 pts)
  • Fato mais incorreto documentado com correção (10 pts)
3
Explorar Geração de Conteúdo
🎨 Criação ⏱ 25 min

Explore as capacidades generativas da IA criando conteúdo em 3 formatos diferentes com qualidade profissional.

📋 Passo a Passo
  1. FORMATO 1 — Texto: use IA para criar um roteiro de 2 minutos para um vídeo do TikTok sobre tecnologia
  2. FORMATO 2 — Estrutura: use IA para criar um plano de negócios de 1 página para um app fictício
  3. FORMATO 3 — Código: use IA para criar pseudocódigo de um algoritmo que você já estudou, mas mais elegante
  4. Para cada formato: avalie a qualidade (1-10) e liste o que teve que corrigir ou melhorar
  5. Reflexão: para qual formato a IA foi mais útil? Em qual precisou de mais intervenção humana?
✅ Rubrica de Avaliação (Total: 100 pts)
  • 3 conteúdos criados com qualidade (30 pts)
  • Avaliação de qualidade com critérios claros (25 pts)
  • Correções documentadas (20 pts)
  • Reflexão sobre utilidade por formato (25 pts)
