📝 Guia de Exercícios — Semana 33
⚡ Sprint 1 4º Bimestre 2 exercícios

Este guia contém as instruções passo a passo, a rubrica de avaliação detalhada e os critérios de qualidade para cada exercício da Semana 33. Siga os passos na ordem indicada e use a rubrica para autoavaliar seu trabalho antes de entregar.

1
Sprint 1 — Produção Intensiva
🔨 Desenvolvimento ⏱ 50 min

Execute o Sprint 1 do projeto usando Pomodoro e Daily Standup para maximizar a produção.

📋 Passo a Passo
  1. INÍCIO: cada membro lê suas tarefas do cronograma e confirma o que deve estar pronto hoje
  2. TÉCNICA POMODORO: configure timer de 25 min. Trabalhe em UMA tarefa sem interrupções
  3. Use IA para criar rascunhos das seções que são sua responsabilidade
  4. Use Bing Image Creator ou Canva para criar imagens do projeto
  5. Após 4 Pomodoros: Daily Standup rápido de 5 min (o que fiz, faço, bloqueios)
  6. FIM: atualize status no cronograma (verde/amarelo/vermelho)
  7. REFLEXÃO: quantos % do Sprint 1 está concluído? O que precisa de atenção?
✅ Rubrica de Avaliação (Total: 100 pts)
  • 60%+ do conteúdo textual criado e revisado (30 pts)
  • Componente de dados completo com gráfico (25 pts)
  • Visual/design consistente definido (20 pts)
  • Cronograma atualizado com status real (15 pts)
  • Reflexão documentada (10 pts)
2
Guia de Estilo do Projeto
🎨 Design ⏱ 15 min

Crie a identidade visual consistente que todos os membros vão usar em todo o projeto.

📋 Passo a Passo
  1. Abra o Canva e crie um documento A4
  2. NOME DO PROJETO em destaque no topo
  3. PALETA: escolha 3 cores principais e mostre os códigos hex
  4. TIPOGRAFIA: 2 fontes (título e corpo), mostre exemplos
  5. ESTILO DE IMAGENS: fotos reais / ilustrações / IA / ícones (escolha um)
  6. LOGOTIPO: crie um simples ou importe o gerado na semana 27
  7. Compartilhe com toda a equipe via link do Canva
✅ Rubrica de Avaliação (Total: 100 pts)
  • Nome do projeto e identidade (20 pts)
  • Paleta com 3 cores e códigos hex (25 pts)
  • Tipografia com exemplos (20 pts)
  • Estilo de imagem definido (20 pts)
  • Compartilhado com a equipe (15 pts)
