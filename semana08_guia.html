📊 Planilhas Avançadas 1º Bimestre 3 exercícios

Este guia contém as instruções passo a passo, a rubrica de avaliação detalhada e os critérios de qualidade para cada exercício da Semana 08. Siga os passos na ordem indicada e use a rubrica para autoavaliar seu trabalho antes de entregar.

1
Fórmula SE e PROCV na Prática
🔧 Fórmulas ⏱ 30 min

Use SE e PROCV para criar um sistema de consulta automática de produtos e classificação de desempenho.

📋 Passo a Passo
  1. Crie uma planilha com 2 abas: 'Produtos' e 'Consulta'
  2. Na aba Produtos: Código | Nome | Preço | Categoria (mínimo 10 produtos)
  3. Na aba Consulta: crie um campo onde o usuário digita o código
  4. Use PROCV para buscar automaticamente: Nome, Preço e Categoria do código digitado
  5. Adicione uma coluna 'Classificação' com SE: se Preço>100 → 'Premium', se >50 → 'Médio', senão → 'Básico'
  6. Teste com 5 códigos diferentes para verificar se funciona
✅ Rubrica de Avaliação (Total: 100 pts)
  • Duas abas criadas e organizadas (15 pts)
  • 10+ produtos cadastrados (15 pts)
  • PROCV funcionando para Nome, Preço e Categoria (30 pts)
  • Classificação com SE aninhado correto (25 pts)
  • 5 testes documentados (15 pts)
2
Formatação Condicional Avançada
🎨 Visual ⏱ 35 min

Aplique regras de formatação condicional para criar um relatório visual de desempenho que destaca padrões automaticamente.

📋 Passo a Passo
  1. Crie uma planilha de desempenho de vendas com 15 linhas de dados
  2. Aplique escala de cores (verde→vermelho) na coluna de vendas
  3. Crie regra: células com valor abaixo da média ficam vermelhas
  4. Crie regra: células com valor acima de 150% da média ficam em dourado
  5. Adicione ícones de setas (↑↓→) baseados em variação percentual
  6. Congele a linha de cabeçalhos (Visualizar → Congelar)
✅ Rubrica de Avaliação (Total: 100 pts)
  • Escala de cores aplicada corretamente (20 pts)
  • Regra 'abaixo da média' funcionando (20 pts)
  • Regra 'acima de 150%' funcionando (20 pts)
  • Ícones de setas aplicados (25 pts)
  • Linha de cabeçalho congelada (15 pts)
3
Gráficos Profissionais
📈 Visualização ⏱ 25 min

Crie 3 tipos diferentes de gráfico para o mesmo dataset e analise qual comunica melhor cada tipo de informação.

📋 Passo a Passo
  1. Use os dados da missão anterior ou crie novos dados de vendas mensais por categoria
  2. Gráfico 1: Barras agrupadas — comparação entre categorias por mês
  3. Gráfico 2: Linhas — evolução temporal de cada categoria
  4. Gráfico 3: Pizza — participação de cada categoria no total anual
  5. Adicione título, rótulos de dados e legenda em cada gráfico
  6. Escreva 1 parágrafo explicando qual gráfico é mais adequado para cada análise
✅ Rubrica de Avaliação (Total: 100 pts)
  • 3 tipos diferentes de gráfico criados (30 pts)
  • Títulos e legendas em todos (20 pts)
  • Rótulos de dados presentes (20 pts)
  • Parágrafo de análise escrito (30 pts)
