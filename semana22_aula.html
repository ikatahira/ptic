Como o ChatGPT Funciona — Por Baixo dos Panos

⚙️
O ChatGPT é um LLM (Large Language Model) — Modelo de Linguagem Grande.

Como foi treinado:
1. Coletou texto de boa parte da internet (Wikipedia, livros, artigos, fóruns)
2. Aprendeu a prever qual palavra vem depois de outra
3. Foi ajustado por humanos para ser útil e seguro (RLHF)

Como gera respostas:
→ Recebe sua pergunta → analisa os padrões aprendidos → gera palavra por palavra, cada uma sendo a mais provável no contexto

Por isso a IA 'alucina': ela não busca a verdade — ela gera o que é estatisticamente provável. Se os dados de treino tinham erros, ela reproduz erros!

Resumo: ChatGPT é um modelo de predição de texto, não uma enciclopédia.

Prompt Engineering — A Arte de Conversar com IA

📝
Prompt é a instrução ou pergunta que você dá para a IA. A qualidade do prompt determina 80% da qualidade da resposta.

Técnicas de Prompt Engineering:

1. Seja específico:
❌ "Me ajuda com português"
✅ "Corrija os erros gramaticais neste texto mantendo meu estilo de escrita"

2. Dê contexto:
✅ "Você é um professor de ETEC. Explique fotossíntese para alunos de 15 anos, usando analogias do cotidiano"

3. Defina o formato:
✅ "Liste 5 vantagens em tópicos curtos, máximo 1 linha cada"

4. Itere: se não ficou bom, refine! "Reescreva mais formalmente" ou "Adicione um exemplo".
⚡ +10 XP
✏️ A instrução ou pergunta que você envia para a IA chama-se:
prompt

🎨

IA Generativa Além do Texto — Imagens, Música e Vídeo

🎨
A IA generativa vai muito além de texto:

🖼 Geração de Imagens:
• Midjourney, DALL-E, Stable Diffusion: geram imagens foto-realistas a partir de descrições
• Exemplo: "fotografia de astronauta tocando violão na Lua, iluminação dourada, estilo cinema"

🎵 Geração de Música:
• Suno, Udio: criam músicas completas com letra, melodia e voz a partir de um prompt
• Exemplo: "música sertaneja universitária sobre tecnologia, ritmo animado, 2 minutos"

🎬 Geração de Vídeo:
• Sora (OpenAI), Runway: criam vídeos realistas a partir de descrições

⚠️ Questões éticas: propriedade intelectual (a IA treinou com arte de artistas sem permissão), deepfakes, desinformação visual.
⚡ +10 XP
✏️ Ferramenta que gera músicas completas a partir de um prompt de texto chama-se:

Verificação de Fatos — Não Acredite em Tudo que a IA Diz

🛡
A IA alucina — inventa informações com a mesma confiança com que diz verdades.

Casos reais de alucinação:
• Advogados citaram jurisprudências inventadas pelo ChatGPT em processos reais
• Estudantes entregaram referências bibliográficas de livros que não existem
• Médicos receberam sugestões de remédios com dosagens perigosas

Como usar IA com segurança:
• ✅ Para rascunhos, brainstorming, aprender conceitos, corrigir textos
• ✅ Sempre verifique informações importantes em fontes confiáveis
• ❌ Não confie em datas, nomes, referências ou dados sem verificar
• ❌ Não use para decisões médicas, legais ou financeiras sem especialista

Regra de ouro: IA é um assistente poderoso, não uma autoridade.
⚡ +10 XP
✏️ Fenômeno onde a IA inventa informações falsas com confiança chama-se:

📝 Prompt Engineering na prática
Domine a arte de criar prompts eficazes para diferentes tarefas.
⏱ 35 min 🤖 ChatGPT/Gemini ⚡ +80 XP
1
TAREFA 1 — Prompt para aprender: peça para a IA explicar um conceito difícil (ex: fotossíntese) de 3 formas: para uma criança de 6 anos, para um estudante de ETEC e para um universitário
2
TAREFA 2 — Prompt para criar: peça para criar um post de Instagram sobre o ETEC com hashtags, máximo 150 caracteres
3
TAREFA 3 — Prompt iterativo: comece com uma pergunta simples, depois peça para a IA refinar a resposta 3 vezes
4
TAREFA 4 — Prompt com persona: 'Você é um recrutador de RH. Avalie este currículo e dê 5 sugestões de melhoria'
5
Para cada tarefa: documente o prompt usado e a qualidade da resposta (nota 1-10)
6
Conclusão: quais técnicas de prompt geraram melhores resultados?

2
🛡 Detectar informações falsas da IA
Treine seu senso crítico verificando afirmações geradas pela IA.
⏱ 25 min 🔍 Verificação ⚡ +70 XP
1
Peça ao ChatGPT: 'Me dê 5 fatos interessantes sobre [escolha um tema]'
2
Para CADA fato: pesquise no Google, Wikipedia ou fonte confiável para verificar
3
Classifique cada um: Verdadeiro / Falso / Parcialmente verdadeiro / Não verificável
4
Se encontrar uma informação falsa ou imprecisa, documente exatamente o que a IA disse e o que é correto
5
Repita com um tema diferente (história, ciência, tecnologia, esportes)
6
Conclusão: qual % das afirmações da IA foi precisa? Esse resultado te surpreendeu?
